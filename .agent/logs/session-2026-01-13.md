## Decisions Made
- **langsci-gb4e format**: The correct format is `\ea[JUDGMENT]{CONTENT}` with:
  - Square brackets for judgment markers (`[]` for grammatical, `[\ungram{}]` for ungrammatical, `[\odd{}]` for odd)
  - Braces for content including `\label{}` and the example text wrapped in `\mention{}`
  - `\z` to close each example
- **Judgment macro placement**: `\ungram{}`, `\marg{}`, `\odd{}` go in square brackets, NOT around the entire content
- Chapter 12 already uses correct format; Chapter 13 was using old linguex-style `\ex.` notation

## Context Learned
- The style guide and style-rules.yaml had incorrect example syntax that didn't match actual chapter usage
- Always check working chapters (like Ch 12) for established patterns before making changes
- The house style uses `\mention{}` for linguistic examples, not bare `\textit{}`
