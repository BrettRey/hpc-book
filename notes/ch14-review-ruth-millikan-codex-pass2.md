**Remaining Concerns**
- From a proper-function standpoint, the chapter still blurs three things: selected function, current use, and analyst utility. “Immune system” language treats correction as if it were automatically proper function (`chapters/chapter14.tex:261`), but some correction practices are culturally recent policing overlays rather than lineage-stabilizing functions.
- The mechanism remains verbal where skeptics will demand formal identifiability. You name variables, but do not specify directional constraints strongly enough to separate “high surprisal with no structural mismatch” from genuine coupling failure in model terms.
- Triadic phenomenology is operationalized, but sign/object/interpretant mapping is still partly stipulative. “Normative correction” can be driven by schooling and prestige effects; the table mentions this confound, but the text still speaks as if corrective convergence directly indexes interpretant-level normativity.
- A few claims remain over-precise for available evidence, e.g. “progressive … is required” (`chapters/chapter14.tex:68`) and near-categorical language around LBE resistance (`chapters/chapter14.tex:275`). These need explicit population/task scope.
- “Syntax gets first pass on it” (`chapters/chapter14.tex:205`) is rhetorically risky in a field-relative framework unless you explicitly say: first pass for syntactic explanatory aims, not ontological ownership.

**Decisive Tests (what now most matters)**
- Orthogonal dissociation design: independently manipulate structural mismatch and processing load; show predicted double dissociation across jolt/resistance/correction.
- Longitudinal update test: repeated exposure should move thresholds for frequency-sensitive constructions but not for conspicuous-absence constraints; pre-register effect sizes and lag structure.
- Proper-function test: distinguish norm-conserving repair that improves producer-consumer coordination from prestige-driven correction that does not. Without this, teleosemantic normativity remains underdetermined.
- Cross-community transfer test: same form in matched contexts across discourse communities should shift detector output in the cohort/register pattern you predict (`chapters/chapter14.tex:334`). This is the strongest near-term discriminator.
