## Overall verdict

**The chapter is now close to publishable and substantially stronger than pass 1.**  
Board consensus is that major conceptual concerns were addressed: mechanism is explicit enough to support test design, triadic phenomenology is operationalized, circularity is reframed as diachronic reflexive equilibrium, and boundary language is materially improved.

**Primary remaining blocker**: empirical identifiability. Reviewers still want explicit bridge principles showing how coupling state, threshold, and processing load are independently constrained.

## Pass-1 to Pass-2 delta (convergent)

1. Mechanism moved from metaphor-led to model-shaped via the five-part detector loop and explicit timescale separation.
2. Phenomenology moved from suggestive to operational through the three-register table (markers, confounds, falsifiers).
3. Acceptability/grammaticality circularity is now mostly defused by the synchronic vs diachronic distinction (“reflexive equilibrium dynamic, not a vicious circle”).
4. Boundary framing improved with the appropriateness-channel architecture; “syntax owns it” style concerns are reduced.
5. Over-precision was partially reduced through scoped language (“appears,” “on current evidence”), but some hard claims remain.

## Convergent concerns (ordered by severity)

1. **Identifiability remains under-specified (dominant concern).**  
Reviewers repeatedly flag that coupling, threshold, and processing/noise can still explain the same judgment patterns unless independently measured.

2. **Model competition needs to be explicit.**  
The chapter should pre-commit to comparisons against processing-only alternatives (surprisal/retrieval/reanalysis) and relevance-based alternatives, ideally with out-of-sample adjudication.

3. **Residual over-strong claims remain.**  
Most recurrent targets: progressive “required” phrasing, near-categorical LBE language, and several universalizing formulations.

4. **Triadic architecture still needs harder failure criteria.**  
The one-factor-collapse risk is acknowledged in text, but reviewers want explicit statistical disconfirmation conditions.

5. **Boundary/ownership language is improved but not fully neutralized.**  
“Syntax gets first pass” is better than ownership language but still reads jurisdictional unless repeatedly framed as purpose-relative.

6. **Secondary but load-bearing requests.**  
Comparative-concept discipline (Croft), developmental staging (Tomasello), and pre-reflective vs reflective separation within normative correction (Gallagher).

## Priority fix set for the next pass

1. Add a compact formal identifiability box (toy model is fine): define latent variables and expected directional effects.
2. Add a short “Identification Strategy” paragraph specifying independent evidence streams (ratings, online processing, repair behavior, community distribution/longitudinal change).
3. Re-scope residual categorical claims to explicit population/task/register bounds.
4. Add an explicit triadic falsifier clause: what empirical result collapses three registers to one factor.
5. Split normative correction into interactional and sociohistorical levels (or explicitly justify why they are bundled).
6. Add one explicit sentence that boundary claims are field-relative to the chapter’s morphosyntactic explanatory aims.

## Decisive test package (highest convergence)

1. Orthogonal factorial design: structural mismatch × processing load (optionally × prescriptive pressure), with preregistered dissociation predictions.
2. Longitudinal entrenchment/change design on an emerging construction with cohort/register structure.
3. Hierarchical model comparison on shared datasets: processing-only vs dual-channel detector vs richer triadic variants.
4. Latent-factor adjudication: test whether jolt/resistance/correction require multiple factors or collapse to one.
5. Community-indexed within-speaker calibration tests (register/audience shifts).

## Recommendation

**Proceed with a targeted final hardening pass, not a structural rewrite.**  
The chapter’s architecture now reads as serious and defensible. The remaining work is methodological tightening and calibration discipline.
