**Strengths**

This is a materially stronger chapter than a typical theory-only revision. In `The stabilizer: a feeling`, the five-part “Mechanism sketch (detector and maintenance loop)” is now explicit enough to be the scaffold for experiments, and the timescale separation (token-time vs entrenchment vs community dynamics) is exactly the right move for avoiding level-mixing.

`Three registers` is also a real upgrade: Table `Operational readout for the three-register proposal` gives observables, confounds, and falsifiers, so triadic phenomenology is no longer pure metaphor. The framing in `Entrenchment in action` (“reflexive equilibrium dynamic, not a vicious circle”) directly addresses the board’s circularity concern and does so in the right temporal way.

Boundary language is much better calibrated in `Grammaticality as a species of appropriateness`: grammaticality is one channel (“morphosyntactic grain”), not the owner of all linguistic normativity. That aligns with the book’s field-relative projectibility stance. Claim calibration has improved too (`on current evidence`, `appears`, scoped prediction/falsifier in the definiteness thread).

**Remaining Concerns**

The mechanism is explicit, but not yet identifiable. Right now, “coupling state,” “decision threshold,” and “noise” can explain the same data; without independent measurement commitments, this risks post hoc fit.

“Structural meaning” is still broad enough to absorb many counterexamples (argument structure, information structure, clause type, force). Skeptics will ask what would *not* count as structural meaning, especially in borderline pragmatics.

Some statements remain too strong for the evidential base: “Every analysis of every construction presupposes…,” “This is exactly what we’d expect…,” and near-categorical rhetoric around LBE persistence. These are easy targets for philosophers and variationists.

Computational falsifiability is implied but not yet sharp. You invoke surprisal and processing channels, but there is no explicit model-comparison agenda against strong alternatives (surprisal-only, grammar-only, relevance-only).

**Priority Fixes (if any)**

1. Add a short subsection with formal commitments: detector output as a function of coupling strength, surprisal/processing load, threshold, and noise.  
2. State how each latent variable will be independently constrained (not inferred from the same judgment data).  
3. Downgrade residual universalizing prose to scoped claims tied to population/register/time.  
4. Recast triadic mapping as a testable hypothesis family, not architecture already secured.

**Decisive Tests (what now most matters)**

Not yet sharp enough for skeptical readers; the chapter needs explicit model contests.

1. Run a preregistered comparison of three models on the same dataset: LM-surprisal-only, categorical grammar-violation, and dual-channel HPC detector.  
2. Use factorial materials crossing structural well-formedness and surprisal (including garden paths and comparative illusions), and fit ratings, RTs, and repair behavior jointly.  
3. Add probe tasks: test whether structural representations (probe/classifier over parser/LM states) explain variance above surprisal alone.  
4. Test satiation longitudinally with hierarchical models: does repeated exposure shift thresholds only for constraints predicted to be entrenchment-sensitive?  
5. Test triadic dissociation directly: early disruption (EEG/eye-tracking), online repair, and delayed normative correction should partially dissociate; if one factor explains all, triadic phenomenology collapses.

If these tests are added as explicit predictions/falsifiers, publishability rises substantially.
