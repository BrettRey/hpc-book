## Overall verdict

**Conceptually strong chapter, but not yet methodologically locked.**

Board consensus: the core move (grammaticality as maintained coupling; intuitions as noisy detector) is strong and publishable. The main blocking risk is not direction, but evidential architecture: mechanism specification, operationalization, and claim calibration need one disciplined pass before this reads as robust rather than metaphor-led.

## Convergent concerns (high agreement)

1. **Mechanism under-specification (near-unanimous)**  
Reviewers repeatedly flagged that `zipper`, `immune system`, `standing wave`, and `spinning top` are vivid but currently do more work than an explicit process model.  
Key pressure point: `chapters/chapter14.tex:43`.

2. **Acceptability vs grammaticality identifiability risk (near-unanimous)**  
Multiple reviews flagged circularity pressure in the “detector tracks what entrenchment constitutes” move.  
Key pressure point: `chapters/chapter14.tex:49`.

3. **Triadic phenomenology not yet operationalized (strong agreement)**  
`jolt/block/correction` is seen as promising, but currently post hoc unless mapped to separable measures/timescales.  
Key pressure points: `chapters/chapter14.tex:78`, `chapters/chapter14.tex:88`.

4. **Over-precise claims without visible method scaffolding (strong agreement)**  
Particular flags: categorical exposure claims and `~3-sigma absence` language.  
Key pressure point: `chapters/chapter14.tex:248`.

5. **Boundary/ownership phrasing risks disciplinary pushback (strong agreement)**  
Repeatedly flagged lines include “syntax owns it” and stronger ontological closure language.  
Key pressure points: `chapters/chapter14.tex:172`, `chapters/chapter14.tex:290`, `chapters/chapter14.tex:314`.

## Distinctive insights from added specialists

1. **Sprouse (methods)**: distinguish criterion shift vs representational shift explicitly; ratings alone are insufficient.
2. **Jaeger/Linzen (modeling)**: require adversarial model comparison against processing-only baselines (surprisal/memory/task noise).
3. **Gallagher (phenomenology)**: separate pre-reflective disruption, reflective judgment, and norm-governed correction as distinct layers, not one generic “feeling.”

## Priority fix set (recommended next pass)

2. **Add an operational table after `§Three registers`**  
Columns: register, timescale, observable marker, confound, falsifier.

3. **Recalibrate strong prose where evidence is suggestive, not settled**  
Target lines: `chapters/chapter14.tex:248`, `chapters/chapter14.tex:314`.

4. **Clarify level separation**  
Token-time parsing effects vs individual entrenchment vs community norm dynamics.

5. **Tighten boundary claim in appropriateness section**  
Keep the broader framing, but add hard inclusion/exclusion tests so grammaticality does not collapse into generic “offness.”

## Decisive test package (cross-review convergence)

1. **Triadic dissociation study**: independently measure jolt/block/correction across garden paths, true structural violations, and register mismatches.
2. **Criterion-vs-representation test**: repeated exposure with transfer to unseen items and production/repair behavior.
3. **Adversarial model comparison**: detector model vs processing-only baseline vs relevance-based account.
4. **Community-indexed calibration test**: cohort/register-structured shifts under entrenchment manipulation.

## Recommendation

**Proceed to a targeted “mechanism + operationalization” revision pass before treating Chapter 14 as settled.**  
The chapter’s conceptual architecture is now broadly endorsed; the requested improvement is evidential hardening, not theoretical redirection.
