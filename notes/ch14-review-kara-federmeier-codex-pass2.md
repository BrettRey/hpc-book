**Strengths**
- The revision materially improves mechanistic clarity. In `§The stabilizer: a feeling`, the boxed “Mechanism sketch (detector and maintenance loop)” finally separates state variables (coupling, processing, detector output, threshold, update history) and distinguishes token-time vs entrenchment vs community dynamics.
- `§Entrenchment in action` now handles acceptability/grammaticality circularity better than before. The line “At a single time-slice, they are separable … Across time … reflexive equilibrium” is the right framing for avoiding a vicious circle.
- The triadic phenomenology is no longer just metaphorical. `§Three registers` plus Table `Operational readout for the three-register proposal` adds observables, confounds, and falsifiers. That is a publishability-level move.
- Boundary language is improved by `§What doesn’t count` and `§Grammaticality as a species of appropriateness`: the chapter now acknowledges multiple “appropriateness channels” rather than treating syntax as sole arbiter.
- Calibration is better in several places (“on current evidence,” “appears to,” “near-categorical in current English evidence”), which helps with philosophical scrutiny.

**Remaining Concerns**
- Mechanism specification is still not fully test-ready. “Detector output: a graded mismatch signal combining coupling state and processing load, plus noise” is conceptually good but computationally underspecified. Without a functional form, parameter identifiability, and update rule, different theories can fit the same data.
- Time-resolved neural predictions remain too generic. Table entries like “EEG perturbation” (100–800 ms) are too broad to discriminate accounts. Skeptical psycholinguists will ask what should happen in early anterior negativity/LAN windows, N400 range, and late positivity/P600 under specific manipulations.
- Triadic phenomenology is staged, but not operationalized enough for adjudication. You need explicit dissociation criteria: what pattern would count as “jolt without resistance,” or “normative correction without jolt,” at participant and item levels.
- A few claims are still over-precise: e.g., LBE “remains categorically rejected regardless of exposure … across tasks, speakers, and presentation modes.” That wording outruns what the literature can safely sustain.
- “Syntax gets first pass on it” is better than ownership language, but the recurring “immune system of the grammar” metaphor can still sound like a global explanatory takeover unless repeatedly scoped as a morphosyntactic channel model.

**Priority Fixes (if any)**
1. Add a minimal formal model (even toy-level): mismatch signal equation, threshold decision function, and entrenchment update equations.
2. Add explicit neural/process predictions with time windows and polarity/direction (not just “EEG perturbation”).
3. Soften the strongest universals (“categorically,” “regardless,” “across speakers”) unless you can cite meta-analytic support.
4. Add a short “adjudication protocol” paragraph: how to classify “is grammatical” when detector output and community norms diverge.

**Decisive Tests (what now most matters)**
1. Orthogonal 2x2 design: structural mismatch (present/absent) × surprisal (high/low), with EEG + eye-tracking + ratings. This directly tests your claim that detector output conflates coupling and processing.
2. Pre-registered triadic dissociation study: map early neural/reading disruption, overt repair behavior, and delayed normative correction to separate latent factors; reject triadic model if one factor explains all variance.
3. Satiation/schema-drift longitudinal test: repeated exposure to marginal constructions with matched controls; model whether threshold shifts, coupling shifts, or both.
4. Community-switch test within speakers (register/discourse manipulation): if detector architecture is community-indexed, correction norms should shift with audience context faster than core parsing signatures.
